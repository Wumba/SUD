Create key on local maschine
- using git bash
$ ssh-keygen -t rsa -b 4096 -C "j.antonius@gmx.net"
- press three time enter (no one needs pass phrases
- start ssh-agent
$ eval $(ssh-agent -s)
- add ssh key to agent
$ ssh-add path/to/ssh/private/key

Add public key to github
$ clip < ~/.ssh/id_rsa.pub
- save it under settings in github

Furhter commands
$ git clone
$ git commit
